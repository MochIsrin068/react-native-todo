 ### Features
 - Add To-Do
 - Mark To-Do
 - Remove To-Do
 - Articles
 - Show User Location

 ## Build Setup

 - Install dependencies
 ```bash
 # (Yarn)
 $ yarn install
 # (npm)
 $ npm install
 ```
 - Run
 ```bash
  $ npx react-native run-android
 ```
